# Placement Prediction System using Machine Learning

## 📌 Project Overview

This project predicts whether a student will get placed or not based on academic performance and employability factors using Machine Learning.

The project uses Logistic Regression for classification and includes data preprocessing, visualization, model training, and evaluation.

---

## 🚀 Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-learn
* Google Colab

---

## 📊 Dataset

The dataset contains student academic and placement-related information such as:

* Gender
* SSC Percentage
* HSC Percentage
* Degree Percentage
* Work Experience
* Specialization
* MBA Percentage
* Placement Status

---

## ⚙️ Machine Learning Workflow

1. Data Loading
2. Data Preprocessing
3. Handling Missing Values
4. Encoding Categorical Data
5. Train-Test Split
6. Logistic Regression Model Training
7. Prediction and Accuracy Evaluation
8. Data Visualization

---

## 🤖 Model Used

* Logistic Regression

---

## 📈 Model Accuracy

The model achieved an accuracy of approximately:

**88.37%**
